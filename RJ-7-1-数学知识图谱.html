第1章 有理数

1.1 正数和负数

1. 正数

定义：大于0的数叫做正数。正数前面的正号可以省略不写。
性质/法则：
  • 正数都大于0
  • 正数前面的正号可以省略不写
  • 正数在数轴上位于原点的右侧
例题类型：计算：(-2)+5

2. 负数

定义：在正数前面加上符号负的数叫做负数。
性质/法则：
  • 负数都小于0
  • 负数前面的负号不可以省略
  • 负数在数轴上位于原点的左侧
例题类型：计算：(-2)+5

3. 0的意义

定义：0既不是正数，也不是负数，它是正数和负数的分界点。
性质/法则：
  • 0是正负数的分界
  • 0可以表示没有
  • 0可以用来表示基准，高于基准的量用正数表示，低于基准的量用负数表示
例题类型：判断：-5<0是否正确

4. 具有相反意义的量

定义：在日常生活中，人们常用正数、负数来表示一对具有相反意义的量。
性质/法则：
  • 必须是同类量，而且是成对出现的
  • 只要求意义相反，不要求数量一定相等
  • 一般把其中一种意义的量规定为正的，用正数表示；把与它意义相反的量规定为负的，用负数表示
例题类型：计算：(-2)+4、判断：5>0是否正确

1.2 有理数

1. 有理数

定义：整数和分数统称为有理数。可以写成分数形式m/n（m、n是整数，n≠0）的数称为有理数。
性质/法则：
  • 有理数包括整数和分数
  • 整数包括正整数、0和负整数
  • 分数包括正分数和负分数
  • 有限小数和无限循环小数都可以化为分数，因此它们都是有理数
例题类型：指出-3、0、2.5、√2中哪些是有理数、判断√4是否为有理数

2. 数轴

定义：规定了原点、正方向和单位长度的直线叫做数轴。
性质/法则：
  • 数轴的三要素：原点、正方向、单位长度（三者缺一不可）
  • 任何一个有理数都可以用数轴上的一个点来表示
  • 数轴上表示的两个数，右边的数总比左边的大
  • 正数大于0，负数小于0，正数大于负数
例题类型：在数轴上表示-2、0、3、比较-4和-1的大小

3. 相反数

定义：只有符号不同的两个数互为相反数。特别地，0的相反数是0。
性质/法则：
  • a和-a互为相反数
  • 在数轴上，表示互为相反数的两个点位于原点的两侧，且到原点的距离相等
  • 互为相反数的两个数之和为0
  • 对任何有理数a，都有|a|=|-a|
例题类型：计算：3×(-4)

4. 绝对值

定义：数轴上表示数a的点与原点的距离叫做数a的绝对值，记作|a|。
性质/法则：
  • 正数的绝对值是它本身
  • 负数的绝对值是它的相反数
  • 0的绝对值是0
  • 任何数的绝对值总是非负数，即|a|≥0
  • 除0外，绝对值为正数的数有两个，它们互为相反数
例题类型：求|-7|、比较-5和-3的大小

1.3 有理数的加减法

1. 有理数加法法则

定义：同号两数相加，取相同的符号，并把绝对值相加；异号两数相加，绝对值相等时和为0，绝对值不等时取绝对值较大的加数的符号，并用较大的绝对值减去较小的绝对值。
性质/法则：
  • 同号两数相加，取相同符号，并把绝对值相加
  • 异号两数相加，绝对值相等时和为0
  • 异号两数相加，绝对值不等时，取绝对值较大的加数的符号，并用较大的绝对值减去较小的绝对值
  • 一个数同0相加，仍得这个数
例题类型：求|-6|、计算：|3-8|

2. 加法运算律

定义：加法交换律：a+b=b+a；加法结合律：(a+b)+c=a+(b+c)。
性质/法则：
  • 加法交换律：两个数相加，交换加数的位置，和不变
  • 加法结合律：三个数相加，先把前两个数相加，或者先把后两个数相加，和不变
  • 灵活运用运算律可以简化计算
例题类型：计算：|3-7|

3. 有理数减法法则

定义：减去一个数，等于加上这个数的相反数。
性质/法则：
  • 减法法则：a-b=a+(-b)
  • 减法运算时注意两变：改变运算符号，改变减数的性质符号
  • 减法运算时注意一不变：被减数与减数的位置不能变换
  • 减法没有交换律
例题类型：计算：-(-9)、求-7的相反数

1.4 有理数的乘除法

1. 有理数乘法法则

定义：两数相乘，同号得正，异号得负，并把绝对值相乘。任何数与0相乘，都得0。
性质/法则：
  • 两数相乘，同号得正，异号得负，并把绝对值相乘
  • 任何数与0相乘，都得0
  • 乘积是1的两个数互为倒数
例题类型：计算：(-2)×3、计算：3×(-4)

2. 乘法运算律

定义：乘法交换律：ab=ba；乘法结合律：(ab)c=a(bc)；乘法分配律：(a+b)c=ac+bc。
性质/法则：
  • 乘法交换律：两个数相乘，交换因数的位置，积不变
  • 乘法结合律：三个数相乘，先把前两个数相乘，或者先把后两个数相乘，积不变
  • 乘法分配律：一个数同两个数的和相乘，等于把这个数分别同这两个数相乘，再把积相加
例题类型：计算：|3-7|

3. 有理数除法法则

定义：除以一个不等于0的数，等于乘这个数的倒数。两数相除，同号得正，异号得负，并把绝对值相除。0除以任何一个不等于0的数，都得0。
性质/法则：
  • 除以一个不等于0的数，等于乘这个数的倒数
  • 两数相除，同号得正，异号得负，并把绝对值相除
  • 0除以任何一个不等于0的数，都得0
  • 0不能作除数
例题类型：求-2的倒数、求3/4的倒数、求|-6|

1.5 有理数的乘方

1. 乘方

定义：求n个相同因数a的积的运算叫做乘方，乘方的结果叫做幂。在a的n次方中，a叫做底数，n叫做指数。
性质/法则：
  • 正数的任何次幂都是正数
  • 负数的奇次幂是负数，负数的偶次幂是正数
  • 0的任何正整数次幂都是0
  • 1的任何次幂都得1
  • -1的偶次幂得1，-1的奇次幂得-1
例题类型：计算：(-2)³、判断：-5<0是否正确

2. 科学记数法

定义：把一个大于10的数表示成a×10的n次方的形式（其中1≤a<10，n是正整数），这种记数法叫做科学记数法。
性质/法则：
  • a的取值范围是1≤a<10
  • n是正整数，等于原数的整数位数减1
  • 科学记数法可以方便地表示很大的数
例题类型：用科学记数法表示：360000

3. 近似数

定义：一个数与准确数相近（比准确数略多或者略少些），这一个数称之为近似数。
性质/法则：
  • 与实际相符的数叫做准确数
  • 与实际接近的数叫做近似数
  • 一个近似数四舍五入到哪一位，就说这个近似数精确到哪一位
  • 从左边第一个非零数字起，到精确到的位数止，所有的数字都叫做这个数的有效数字
例题类型：计算：(-2)+5

第2章 整式的加减

2.1 整式

1. 单项式

定义：由数与字母的积组成的代数式叫做单项式。单独的一个数或一个字母也是单项式。
性质/法则：
  • 单项式中的数字因数叫做这个单项式的系数
  • 一个单项式中，所有字母的指数的和叫做这个单项式的次数
  • 单独的一个非零数的次数是0
例题类型：判断3x²y是否为单项式、指出-4x³的系数和次数

2. 多项式

定义：几个单项式的和叫做多项式。
性质/法则：
  • 在多项式中，每个单项式叫做多项式的项
  • 不含字母的项叫做常数项
  • 多项式里次数最高项的次数，叫做这个多项式的次数
例题类型：判断x²+2x-1是否为多项式、指出3x²-5x+2的项和次数

3. 整式

定义：单项式与多项式统称整式。
性质/法则：
  • 整式包括单项式和多项式
  • 整式中分母不含字母
例题类型：判断：-5<0是否正确

2.2 整式的加减

1. 同类项

定义：所含字母相同，并且相同字母的指数也相同的项叫做同类项。几个常数项也是同类项。
性质/法则：
  • 判断同类项的条件：所含字母相同；相同字母的指数也相同
  • 同类项与系数无关，与字母的排列顺序无关
  • 几个常数项也是同类项
例题类型：判断3x²与-5x²是否为同类项、判断：-5<0是否正确

2. 合并同类项

定义：把多项式中的同类项合并成一项，叫做合并同类项。
性质/法则：
  • 合并同类项后，所得项的系数是合并前各同类项的系数的和，且字母连同它的指数不变
  • 如果两个同类项的系数互为相反数，合并同类项后结果为0
  • 不是同类项的不能合并
例题类型：多项式2x²-3x+1的次数是、计算：-(-9)、求-7的相反数

3. 去括号法则

定义：如果括号外的因数是正数，去括号后原括号内各项的符号与原来的符号相同；如果括号外的因数是负数，去括号后原括号内各项的符号与原来的符号相反。
性质/法则：
  • 括号前是'+'号，把括号和它前面的'+'号去掉后，原括号里各项的符号都不改变
  • 括号前是'-'号，把括号和它前面的'-'号去掉后，原括号里各项的符号都要改变
  • 去括号时，要连同括号前面的符号一起去掉
例题类型：解方程：2(x+1)=8、判断3x²y与-2x²y是否为同类项、解方程：3(x-2)=9

4. 整式的加减运算

定义：一般地，几个整式相加减，如果有括号就先去括号，然后再合并同类项。
性质/法则：
  • 整式加减的步骤：先去括号，再合并同类项
  • 整式加减的结果仍是整式
  • 注意符号的变化
例题类型：计算：(3a-2b)+(a+b)

第3章 一元一次方程

3.1 从算式到方程

1. 方程

定义：含有未知数的等式叫做方程。
性质/法则：
  • 方程必须含有未知数
  • 方程必须是等式
  • 方程中的未知数可以是x、y、z等字母
例题类型：判断：-5<0是否正确

2. 一元一次方程

定义：只含有一个未知数（元），未知数的次数都是1，等号两边都是整式，这样的方程叫做一元一次方程。
性质/法则：
  • 只含有一个未知数
  • 未知数的次数都是1
  • 等号两边都是整式
  • 一元一次方程的一般形式：ax+b=0（a≠0）
例题类型：判断：-5<0是否正确

3. 方程的解

定义：使方程中等号左右两边相等的未知数的值叫做方程的解。
性质/法则：
  • 方程的解是使方程成立的未知数的值
  • 检验一个数是否为方程的解，只需将该数代入方程，看等号两边是否相等
例题类型：解方程：2x=10、解方程：x+3=7

4. 等式的性质

定义：等式两边加（或减）同一个数（或式子），结果仍相等；等式两边乘同一个数，或除以同一个不为0的数，结果仍相等。
性质/法则：
  • 等式性质1：等式两边加（或减）同一个数（或式子），结果仍相等。如果a=b，那么a±c=b±c
  • 等式性质2：等式两边乘同一个数，或除以同一个不为0的数，结果仍相等。如果a=b，那么ac=bc；如果a=b（c≠0），那么a/c=b/c
例题类型：计算：|3-7|

3.2 解一元一次方程（一）——合并同类项与移项

1. 移项

定义：把等式一边的某项变号后移到另一边，叫做移项。
性质/法则：
  • 移项要变号
  • 移项的依据是等式性质1
  • 移项的目的是将含未知数的项移到等号的一边，常数项移到等号的另一边
例题类型：解方程：x-5=3

2. 合并同类项解方程

定义：将方程中同类项合并，化为ax=b（a≠0）的形式，再求解。
性质/法则：
  • 合并同类项可以简化方程
  • 合并后化为ax=b的形式
  • 最后系数化为1求得解
例题类型：合并：3a+5a、判断3x²y与-2x²y是否为同类项、解方程：3x+2=11

3.3 解一元一次方程（二）——去括号与去分母

1. 去括号解方程

定义：解含有括号的一元一次方程时，先去括号，再移项、合并同类项、系数化为1。
性质/法则：
  • 按照去括号法则去掉方程中的括号
  • 注意括号前是负号时，括号内各项要变号
  • 去括号后按常规步骤解方程
例题类型：合并：3a+5a、解方程：3x+2=11

2. 去分母解方程

定义：解含有分母的一元一次方程时，先去分母，再去括号、移项、合并同类项、系数化为1。
性质/法则：
  • 去分母时，方程两边同乘各分母的最小公倍数
  • 分子是多项式时，去分母后要加括号
  • 不含分母的项也要乘最小公倍数
例题类型：解方程：x+3=7、解方程：3(x-2)=9、解方程：2(x+1)=8

3.4 实际问题与一元一次方程

1. 列一元一次方程解应用题

定义：用一元一次方程解决实际问题的基本步骤：审、设、列、解、验、答。
性质/法则：
  • 审：审题，分析题意，找出已知量和未知量
  • 设：设未知数，可以直接设，也可以间接设
  • 列：根据等量关系列出方程
  • 解：解方程，求出未知数的值
  • 验：检验解是否符合实际意义
  • 答：写出答案
例题类型：解方程：x+3=7

第4章 几何图形初步

4.1 几何图形

1. 立体图形

定义：有些几何图形（如长方体、正方体、圆柱、圆锥、球等）的各部分不都在同一平面内，它们是立体图形。
性质/法则：
  • 立体图形的各部分不都在同一平面内
  • 常见的立体图形有：长方体、正方体、圆柱、圆锥、球、棱柱、棱锥等
  • 棱柱：n棱柱有两个底面，n个侧面，共（n+2）个面；3n条棱；2n个顶点
例题类型：计算：(-2)+5

2. 平面图形

定义：有些几何图形（如线段、角、三角形、长方形、圆等）的各部分都在同一平面内，它们是平面图形。
性质/法则：
  • 平面图形的各部分都在同一平面内
  • 常见的平面图形有：线段、角、三角形、长方形、正方形、圆等
例题类型：计算：(-2)+5

3. 展开图

定义：有些立体图形是由一些平面图形围成的，将它们的表面适当剪开，可以展开成平面图形，这样的平面图形称为相应立体图形的展开图。
性质/法则：
  • 正方体的展开图有11种
  • 口诀：中间四个面，上下各一面；中间三个面，一二隔河见；中间二个面，楼梯天天见；中间没有面，三三连一线
  • 田、凹应弃之
例题类型：计算：(-12)÷3、比较-5和-3的大小、求-8的相反数

4. 点、线、面、体

定义：几何体也简称体，包围着体的是面，面与面相交的地方形成线，线与线相交的地方是点。
性质/法则：
  • 点动成线，线动成面，面动成体
  • 点是几何图形中最基本的图形
  • 面分为平面和曲面，线分为直线和曲线
例题类型：过两点能画几条直线、圆柱的侧面展开图是什么

4.2 直线、射线、线段

1. 直线

定义：经过两点有一条直线，并且只有一条直线。简单说成：两点确定一条直线。
性质/法则：
  • 直线没有端点，向两方无限延伸
  • 经过两点有且只有一条直线（两点确定一条直线）
  • 直线可以用直线上的两个点来表示，如直线AB
例题类型：过两点能画几条直线、线段AB=5cm，BC=3cm，求AC

2. 射线

定义：直线上一点和它一旁的部分叫做射线。
性质/法则：
  • 射线有一个端点，向一方无限延伸
  • 射线可以用端点和射线上另一点来表示，如射线OA
例题类型：线段AB=5cm，BC=3cm，求AC、画射线OA

3. 线段

定义：直线上两点及两点间的部分叫做线段。
性质/法则：
  • 线段有两个端点，不向任何一方延伸
  • 线段可以用它的两个端点来表示，如线段AB
  • 两点之间，线段最短
  • 连接两点的线段的长度，叫做这两点的距离
例题类型：线段AB=5cm，BC=3cm，求AC、过两点能画几条直线、比较两条线段的长短

4. 线段的中点

定义：点M把线段AB分成相等的两条线段AM与MB，点M叫做线段AB的中点。
性质/法则：
  • 如果点M是线段AB的中点，那么AM=MB=1/2AB
  • 反之，如果AM=MB=1/2AB，那么点M是线段AB的中点
例题类型：计算：3×(-4)

4.3 角

1. 角

定义：有公共端点的两条射线组成的图形叫做角。
性质/法则：
  • 角的顶点是两条射线的公共端点
  • 角的两边是两条射线
  • 角也可以看作由一条射线绕着它的端点旋转而形成的图形
例题类型：旋转中心是、一个平角等于多少度

2. 角的度量

定义：度、分、秒是常用的角的度量单位。把一个周角360等分，每一份就是1度的角，记作1°。
性质/法则：
  • 1°=60′，1′=60″
  • 1周角=360°，1平角=180°，1直角=90°
  • 锐角：大于0°小于90°的角；直角：等于90°的角；钝角：大于90°小于180°的角
例题类型：一个平角等于多少度、计算：45°+30°、将30.5°转换为度分

3. 角的平分线

定义：从一个角的顶点出发，把这个角分成相等的两个角的射线，叫做这个角的平分线。
性质/法则：
  • 如果OC是∠AOB的平分线，那么∠AOC=∠BOC=1/2∠AOB
  • 反之，如果∠AOC=∠BOC=1/2∠AOB，那么OC是∠AOB的平分线
例题类型：判断：-5<0是否正确

4. 余角和补角

定义：如果两个角的和等于90°（直角），就说这两个角互为余角；如果两个角的和等于180°（平角），就说这两个角互为补角。
性质/法则：
  • 同角（等角）的余角相等
  • 同角（等角）的补角相等
例题类型：计算：3×(-4)

5. 方位角

定义：方位角是用角度和方向来表示位置的角。
性质/法则：
  • 通常以正北或正南方向为基准
  • 表示为北偏东、北偏西、南偏东、南偏西等形式
例题类型：一个平角等于多少度、计算：45°+30°
